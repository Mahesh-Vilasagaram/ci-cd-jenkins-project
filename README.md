# CI/CD Pipeline using GitHub and Jenkins

## Overview
This project demonstrates an automated CI/CD pipeline where code changes in GitHub trigger Jenkins builds and deployments.

## Tools Used
- GitHub
- Jenkins
- AWS EC2
- Linux
- Jenkinsfile (Pipeline as Code)

## Pipeline Flow
GitHub Push → Jenkins Webhook → Build → Deploy

## Key Highlights
- Automated deployments
- Pipeline defined using Jenkinsfile
- Reduced manual intervention
